# Install

This is standard flask, use
```
pip install flask
```

Tell Flask where to start 

### Powershell
```
$env:FLASK_APP ="server/flaskr"
```

### bash 
```
$ export FLASK_ENV=development
$ flask run --host=0.0.0.0
```

### Running flask 
```flask run```
